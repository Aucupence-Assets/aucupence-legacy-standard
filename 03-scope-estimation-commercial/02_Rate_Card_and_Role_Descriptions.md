# Standard Role Descriptions & Rate Card – Reference

## 1. Core Roles

- **Engagement Director**  
  Accountable for executive relationship, commercials, and risk posture.

- **Project Manager / Scrum Master**  
  Responsible for day-to-day coordination, sprint planning, and stakeholder communication.

- **Solution Architect**  
  Owns end-to-end solution integrity, approves key design decisions and non-functional trade-offs.

- **Tech Lead(s)**  
  Responsible for technical decisions within their domain, code quality, and mentoring team members.

- **Developers (Backend / Frontend / Full-stack)**  
  Deliver production-grade code, participate in design and code reviews.

- **QA Engineers (Manual + Automation)**  
  Design and execute test cases, maintain automated regression packs.

- **DevOps / SRE**  
  Build and maintain deployment pipelines, monitoring, reliability practices.
